# Discord Channel Crawler

A Chrome extension that crawls Discord channel messages and stores them in a SQLite database.

## Features

- Automatically crawls Discord channel messages by scrolling through the chat
- Stores messages, channels, users, and reactions in IndexedDB
- Export database to SQLite file format
- View statistics (message count, channels crawled, last crawled time)
- Browse crawled data with filtering by channel
- Shows channel names and usernames in data viewer
- Tracks message reactions

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked"
4. Select the extension directory
